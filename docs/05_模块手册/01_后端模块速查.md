## app/infra/config/

```python
from app.infra.config import settings
settings.JWT_SECRET_KEY       # str
settings.DOCKER_STORAGE_PATH  # str
settings.PORT                 # int，默认 8000
settings.DB_PATH              # str
settings.APP_ENV              # str，'development' | 'production'（默认 'production'）
```

## app/infra/security/crypto.py

```python
crypto = get_crypto_manager()  # 懒初始化全局单例

crypto.encrypt_api_key(plain)     # → 密文
crypto.decrypt_api_key(cipher)    # → 明文，失败返回空字符串
crypto.get_password_hash(pwd)     # → bcrypt 哈希
crypto.verify_password(plain, hashed)  # → bool
crypto.create_access_token(username)   # → JWT（7天）
crypto.verify_token(token)        # → username 或 None
crypto.init_admin(username, pwd)  # 仅允许执行一次
crypto.authenticate(username, pwd) # → bool
```

```python
    movies: int = 0
```

### domain_system.py

```python
class PathConfig(BaseModel):
    id: Optional[int]
    type: str        # 'download' | 'library'
    path: str
    category: str    # 'movie' | 'tv' | 'mixed'
    enabled: bool

class SettingsConfig(BaseModel):
```

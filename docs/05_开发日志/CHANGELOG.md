## [V1.0.0] - 2026-03-10

### 新增功能

- **JWT 认证体系**：完整登录鉴权链路，含 `AuthGuard` 路由守卫、Token 持久化、自动跳转登录页
- **正则实验室（RegexLab）**：可视化正则规则调试工具，支持实时测试与规则保存
- **正则军火库**：内置常用媒体文件名清洗规则集，支持自定义扩展
- **i18n 国际化**：前端多语言支持（`frontend/lib/i18n.ts` + `useLanguage` Hook）
- **数据库自动迁移**：启动时自动检测并执行 Schema 变更，无需手动 migrate
- **配置重置功能**：支持一键恢复默认配置
- **迷你日志面板（MiniLog）**：实时 SSE 日志流展示组件
- **系统监控面板（SystemMonitor）**：CPU / 内存 / 任务状态实时监控
- **寻猎者 V9.2**：TMDB 指纹对齐升级，提升元数据匹配准确率
- **N8N 归档全链路**：Servarr 下载器与硬链接归档组织器打通
- **防重与冷却机制**：任务重试限流，防止重复触发

### 视觉 / UI

- **Dashboard 装甲化重构**：70/30 非对称布局，赛博朋克霓虹主题全面落地
- **全局 CSS 变量系统**：`globals.css` 统一色彩、间距、阴影变量
- **海报墙视图（MediaWall）**：媒体库海报网格展示，支持分页
- **媒体工具栏（MediaToolbar）**：扫描 / 刮削 / 归档一键触发
- **赛博粒子背景（CyberParticles）**：Canvas 粒子动画背景

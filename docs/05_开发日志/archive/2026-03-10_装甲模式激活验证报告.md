## 四、技术实现亮点

### 1. 双重边框实现原理

通过 `box-shadow` 的 `inset` 关键字，实现内部边框效果：

```css
box-shadow: 
  inset 0 0 0 3px var(--cyber-cyan),           /* 内部青色边框 */
  inset 3px 3px 0 4px rgba(0, 0, 0, 0.4),      /* 内部深色间隙 */
  12px 12px 0px rgba(249, 240, 2, 0.25),       /* 外部实体投影 */
  0 0 25px rgba(249, 240, 2, 0.35);            /* 外部发光 */
```

**技术要点**:
- 第1层 `inset` 创建内部青色边框
- 第2层 `inset` 创建内部深色间隙，模拟装甲板层叠
- 第3层创建外部实体投影，增加立体感
- 第4层创建外部发光，增强霓虹效果

### 2. 斜角裁剪实现原理

通过 `clip-path: polygon()` 实现不规则切角：

```css
clip-path: polygon(
  0 0,                          /* 左上角 */
  calc(100% - 18px) 0,          /* 右上角前 18px */
  100% 18px,                    /* 右上角斜切 */
  100% calc(100% - 35px),       /* 右下角前 35px */
  calc(100% - 35px) 100%,       /* 右下角斜切 */
  0 100%                        /* 左下角 */
);
```

### 3. 霓虹呼吸光晕实现原理

通过 CSS 动画实现光晕呼吸效果：

```css
@keyframes input-glow-pulse {
  0%, 100% {
    box-shadow: 
      0 0 0 2px var(--yellow-color-opacity),
      0 0 15px rgba(249, 240, 2, 0.5),
      0 0 30px rgba(249, 240, 2, 0.25);
  }
  50% {
    box-shadow: 
      0 0 0 2px var(--yellow-color-opacity),
      0 0 20px rgba(249, 240, 2, 0.7),
      0 0 40px rgba(249, 240, 2, 0.35);
  }
}
```

## 五、后续优化建议

### 1. 组件级应用
建议在以下组件中应用装甲化通用类：
- MediaTable.tsx: 表格行应用 `.table-row-armor`
- SettingsHub.tsx: 设置面板应用 `.panel-armor`
- AIChat.tsx: 对话框应用 `.modal-armor`
- 所有按钮: 应用 `.btn-armor` 或 `.cyber-button`
- 所有输入框: 应用 `.input-armor`

### 2. 响应式优化
在小屏幕设备上，建议适当减小装甲边框厚度：

```css
@media (max-width: 768px) {
  .glass-effect {
    border: 6px solid var(--cyber-yellow) !important;
    border-left: 18px solid var(--cyber-yellow) !important;
  }
}
```

### 3. 性能优化
- `backdrop-filter: blur(10px)` 在低端设备上可能影响性能
- 大量 `box-shadow` 叠加可能影响渲染性能
- 建议在移动端减少层数或添加降级方案

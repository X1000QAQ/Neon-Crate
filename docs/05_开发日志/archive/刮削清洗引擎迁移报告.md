## 🎯 完成内容

### 1. 核心代码实现

**文件路径**: `backend/app/services/scraper/cleaner.py`

**核心类**: `MediaCleaner`

**代码行数**: 393 行

**核心方法**:
- `clean_name(filename: str) -> str`: 剥离所有杂质标签，返回纯净名称
- `extract_year(filename: str) -> Optional[int]`: 提取年份
- `extract_season_episode(filename: str) -> Tuple[Optional[int], Optional[int]]`: 提取季和集
- `is_tv_show(filename: str) -> bool`: 判断是否为剧集
- `is_advertisement(filename: str) -> bool`: 判断是否为广告
- `clean_and_extract(filename: str) -> dict`: 一站式处理

### 2. 正则表达式迁移

从 N8N 的 4 个工作流文件中提取并翻译了以下核心正则：

| 类型 | JavaScript (N8N) | Python (cleaner.py) | 说明 |
|------|------------------|---------------------|------|
| 分辨率过滤 | `/1080p\|720p\|4k/gi` | `r'\b(2160p\|1080p\|720p\|4k\|...)\b'` | 过滤分辨率标签 |
| 编码过滤 | `/x264\|x265\|HEVC/gi` | `r'\b(x264\|x265\|H\.264\|...)\b'` | 过滤编码格式 |
| 压制组过滤 | `/\[.*?\]/g` | `r'\[[^\]]*?(?:Raws?\|...)[^\]]*?\]'` | 过滤压制组标签 |
| 年份提取 | `/\((\d{4})\)/` | `r'[\(\[\.\s]+(19\d{2}\|20\d{2})[\)\]\.\s]+'` | 提取年份 |
| 季集提取 | `/[Ss](\d+)[Ee](\d+)/` | `r'[Ss](\d{1,2})[Ee](\d{1,3})'` | 提取季集信息 |
| 动漫格式 | `/-\s*(\d{2,4})(?=\s*\[)/` | `r'[-\s](\d{2,4})(?=\s*\[)'` | 动漫番剧格式 |

### 2. 非对称重型装甲容器

#### 主装甲：厚实的黄色边框
```css
.glass-effect {
  border: 10px solid var(--cyber-yellow);
  border-left: 20px solid var(--cyber-yellow);
  border-top: 12px solid var(--cyber-yellow);
}
```

#### 几何裁剪：非对称斜角
```css
clip-path: polygon(
  30px 0,                    /* 左上角斜切 */
  0 30px,
  0 100%,
  calc(100% - 40px) 100%,    /* 右下角斜切 */
  100% calc(100% - 40px),
  100% 0
);
```

#### 右侧能量点缀（关键特性）
```css
.glass-effect::after {
  content: "";
  right: 8px;
  width: 5px;
  height: 70%;
  background: linear-gradient(
    180deg,
    transparent 0%,
    var(--neon-color) 10%,
    var(--cyber-cyan) 50%,
    var(--neon-color) 90%,
    transparent 100%
  );
  box-shadow: 0 0 10px var(--cyber-cyan);
  animation: energy-pulse 2s ease-in-out infinite;
}
```

**视觉效果**:
- ✅ 右侧垂直青色能量条（5px × 70%）
- ✅ 微弱外发光（模拟能量流动）
- ✅ 脉动动画（2s 循环）

## 🎯 技术红线遵守情况

### ✅ 青色只能作为右侧点缀色

所有青色元素均满足以下条件：
- ✅ 使用 `::after` 伪元素实现
- ✅ 位于右侧（`right: 4-12px`）
- ✅ 宽度控制在 3-6px
- ✅ 不影响主体布局

### ✅ 黄色装甲主体视觉位不被抢占

- ✅ 主边框：10px 黄色
- ✅ 左侧装甲：20px 黄色
- ✅ 顶部装甲：12px 黄色
- ✅ 实体投影：黄色（12px 12px）
- ✅ 外发光：黄色（25px）

**对比数据**:
- 黄色边框厚度：10-20px
- 青色点缀宽度：3-6px
- **比例**: 黄色是青色的 3-6 倍

### ✅ Advent Pro 与 Hacked 的优雅对比

| 字体 | 应用场景 | 字符间距 | 视觉特征 |
|------|---------|---------|---------|
| Advent Pro | 全局（标题、正文、按钮） | 0.5-1.5px | 科技感、优雅 |
| Hacked | 密码框及标签 | 2px | 黑客硬核、终端感 |

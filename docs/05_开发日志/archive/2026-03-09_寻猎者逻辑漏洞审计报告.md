**漏洞分析**:

#### 🔴 漏洞点 1: 无脑选择 `results[0]`

```python
# 第 167 行
series_data = lookup_results[0]  # ❌ 直接取第一项，不做任何校验
```

**问题描述**:
- Sonarr API 返回的搜索结果是按照 **TVDB 的相关性算法** 排序的
- 当搜索 `"庆余年2"` 时，TVDB 可能返回：
  1. 《乱马 1/2》（因为包含 "1/2" 数字匹配）
  2. 《庆余年 第二季》（真正的目标）
  3. 其他包含 "2" 的剧集
- 代码直接取 `[0]`，导致选中错误的《乱马 1/2》

#### 🔴 漏洞点 2: 缺乏 Title 相似度校验

**现状**: 代码中 **完全没有** 任何相似度比对逻辑

**应有的逻辑**:
```python
# ❌ 当前代码：无任何校验
series_data = lookup_results[0]

# ✅ 应有的逻辑：相似度校验
from difflib import SequenceMatcher

best_match = None
best_score = 0.0

for result in lookup_results:
    result_title = result.get("title", "").lower()
    score = SequenceMatcher(None, title.lower(), result_title).ratio()
    
    if score > best_score:
        best_score = score
        best_match = result

if best_score < 0.6:  # 相似度阈值
    return {"success": False, "msg": f"未找到与「{title}」足够相似的剧集"}

series_data = best_match
```

```python
from difflib import SequenceMatcher

def _find_best_match(self, search_term: str, results: list) -> tuple:
    """
    从搜索结果中找到最佳匹配项
    
    Args:
        search_term: 用户搜索的片名
        results: Sonarr/Radarr 返回的搜索结果列表
        
    Returns:
        tuple: (best_match_data, similarity_score)
    """
    if not results:
        return None, 0.0
    
    best_match = None
    best_score = 0.0
    
    search_lower = search_term.lower().strip()
    
    for result in results:
        title = result.get("title", "").lower().strip()
        
        # 计算相似度
        score = SequenceMatcher(None, search_lower, title).ratio()
        
        # 如果完全匹配，直接返回
        if score == 1.0:
            return result, score
        
        # 记录最高分
        if score > best_score:
            best_score = score
            best_match = result
    
    return best_match, best_score

# 在 add_series 和 add_movie 中替换：
# series_data = lookup_results[0]  # ❌ 旧代码
series_data, similarity = self._find_best_match(search_term, lookup_results)

# 相似度阈值校验
if similarity < 0.6:
    logger.warning(f"⚠️ [Sonarr] 最佳匹配相似度过低: {similarity:.2f}")
    return {
        "success": False, 
        "msg": f"未找到与「{title}」足够相似的剧集（最佳匹配: {series_data.get('title', 'Unknown')}，相似度: {similarity:.0%}）",
        "data": {}
    }

logger.info(f"✅ [Sonarr] 找到最佳匹配: {series_data.get('title', 'Unknown')} (相似度: {similarity:.0%})")
```

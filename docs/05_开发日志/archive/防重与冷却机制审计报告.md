```python
def check_task_exists_by_path(self, file_path: str) -> bool:
    """检查指定路径的任务是否已存在"""
    with self.db_lock:
        conn = sqlite3.connect(self.db_path)
        cursor = conn.execute(
            "SELECT 1 FROM tasks WHERE path = ? LIMIT 1",
            (file_path,)
        )
        result = cursor.fetchone() is not None
        conn.close()
        return result
```

### ✅ 审计结论

**状态**: 已完整实现  
**对比结果**: Python 后端的防重逻辑与 N8N 工作流完全一致  

**防御机制**:
1. 扫描前从数据库加载所有已处理路径
2. 扫描时对每个文件执行 `check_task_exists_by_path()` 检查

- 线程安全（`db_lock` 保护）

## ✅ 第二道防线：字幕姬冷却机制 (Subtitle Cooldown Mechanism)

### N8N 原有逻辑

```javascript
// N8N 字幕姬 V13.3 - Pre-Processor 节点
const RETRY_DAYS = 7; // 冷却天数

for (const f of files) {
    // 规则 A: 检查本地是否已有字幕
    if (hasAiSub) continue;
    
    // 规则 B: 检查数据库状态（含冷却期）
    const record = subDb[fullPath];
    if (record) {
        if (record.status === 'SUCCESS') continue;
        
        // 🔥 核心防御：检查 "未找到中文" 的冷却期
        if (record.status === 'NOT_FOUND_ZH') {
            const lastTime = new Date(record.time).getTime();
            const daysDiff = (now - lastTime) / (1000 * 3600 * 24);
            if (daysDiff < RETRY_DAYS) continue; // 还在冷却期内，跳过
        }
    }
}
```

**防御目标**: 如果上次搜索字幕返回"未找到中文"（`NOT_FOUND_ZH`），必须有一个 7 天的冷却期，期间不再对该文件发起字幕搜索请求。

def get_tasks_needing_subtitles(self) -> List[Dict[str, Any]]:
    """
    获取需要字幕的任务列表（含 7 天冷却期过滤）
    
    条件：
    1. status = 'archived'（已归档）
    2. tmdb_id 不为空（已刮削）
    3. sub_status 为 'pending' 或 'failed' 或 NULL
    4. 或 sub_status = 'missing' 且距离上次检查已超过 7 天（冷却期）
    
    N8N 对齐逻辑：
    - 如果上次搜索返回"未找到中文"（missing），必须等待 7 天冷却期
    - 冷却期内不再对该文件发起字幕搜索请求
    """
    with self.db_lock:
        conn = sqlite3.connect(self.db_path)
        
        # 计算 7 天前的时间戳
        from datetime import datetime, timedelta
        cooldown_date = (datetime.now() - timedelta(days=7)).strftime("%Y-%m-%d %H:%M:%S")
        
        cursor = conn.execute("""
            SELECT id, path, file_name, type, tmdb_id, imdb_id, target_path, sub_status
            FROM tasks
            WHERE status = 'archived'
              AND tmdb_id IS NOT NULL 
              AND tmdb_id != ''
              AND (
                  sub_status IN ('pending', 'failed') 
                  OR sub_status IS NULL
                  OR (sub_status = 'missing' AND (last_sub_check IS NULL OR last_sub_check < ?))
              )
            ORDER BY created_at ASC
        """, (cooldown_date,))

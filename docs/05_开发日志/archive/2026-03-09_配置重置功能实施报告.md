## 一、需求背景

### 1.1 问题描述

用户在修改 AI 规则和正则清洗规则后，可能因配置错误导致系统功能异常。特别是在以下场景：

1. **配置残留问题**: 旧版本配置与新版本不兼容，导致前端界面空白
2. **用户误操作**: 错误修改关键配置导致系统无法正常工作
3. **版本升级**: 系统升级后需要重置为新版本的标准配置

### 1.2 解决方案

为设置界面增加"重置为工业级默认值"功能，支持：
- AI 规则重置（ai_persona, expert_archive_rules, master_router_rules）
- 正则清洗规则重置（filename_clean_regex，从 MediaCleaner 自动提取 15 条规则）

## 二、技术实现

### 2.1 后端实现

#### 2.1.1 数据库管理器新增方法

**文件**: `backend/app/infra/database/db_manager.py`

**新增方法**: `reset_settings_to_defaults(self, target: str)`

**参数逻辑**:
- `target == "ai"`: 重置 AI 规则为 V2.1 去 N8N 化标准 Prompt
- `target == "regex"`: 从 MediaCleaner 重新抓取 15 条正则并覆盖

**核心特性**:
1. 强制覆盖配置为工业级默认值
2. 自动调用 `_save_config()` 持久化到磁盘
3. 严格遵守标准数据契约（clean_name, chinese_title, original_title, year, type, season, episode）

#### 2.1.2 系统接口新增端点

**文件**: `backend/app/api/v1/endpoints/system.py`

**新增接口**: `POST /api/v1/settings/reset`

**请求体**:
```json
{
  "target": "ai" | "regex"
}
```

**响应体**:
```json
{
  "success": true,
  "message": "[OK] AI 配置已重置为工业级默认值"
}
```

**日志规范**: 
- 打印格式: `[INFO] [API] 触发配置重置: {target}`
- 严禁使用 Emoji（遵守 Windows 环境避坑指南）

### 2.2 前端实现

#### 2.2.1 PersonaSettings 组件

**文件**: `frontend/components/settings/PersonaSettings.tsx`

**新增功能**:
1. 增加"重置为默认值"按钮（橙色样式）

## 三、默认配置内容

### 3.1 AI 规则默认值

#### 3.1.1 AI Persona（智能中枢人格）

```
你是一个负责调度工具的智能中枢。你是行动派，严禁复读，每一条指令都视为全新任务。

核心原则：
1. 无工具不开口：严禁在未调用工具的情况下直接回复。必须先看到工具调用的结果，才能说话。
2. 强制动作：每一次用户的输入，无论是否与上一句重复，都必须视为一个全新的、独立的指令。
3. 传声筒原则：调用工具时，参数必须一字不差地复制用户的原话。严禁总结、严禁提取关键词。
```

#### 3.1.2 Expert Archive Rules（归档专家规则）

包含完整的标准数据契约定义，确保输出字段为：
- `clean_name`: 最通用、最纯净的作品名称（不含年份）
- `chinese_title`: 豆瓣/B站官方中文译名
- `original_title`: 原产地名称
- `year`: 年份（4位数字字符串）
- `type`: 类型（movie/tv/IGNORE）
- `season`: 季数（整数）
- `episode`: 集数（整数）

#### 3.1.3 Master Router Rules（总控路由规则）

包含标准数据契约统一键名规范和铁腕控制逻辑。

### 3.2 正则清洗规则默认值

1. 分辨率标签过滤

3. 方括号技术标签过滤
4. 花括号标签过滤
5. 广告词过滤
6. 音频/视频特性标签过滤
7. 语言标签过滤
8. 制作组后缀过滤
9. 年份提取
10. 季集信息提取-S01E01格式
11. 季集信息提取-Season格式
12. 季集信息提取-1x01格式
13. 季集信息提取-EP01格式
14. 季集信息提取-中文格式
15. 动漫番剧特殊格式

## 六、使用指南

### 6.1 重置 AI 规则

1. 进入"设置" → "AI 规则设置"（Persona 标签页）
2. 点击底部橙色"重置为默认值"按钮
3. 确认弹窗提示
4. 等待页面自动刷新

### 6.2 重置正则清洗规则

1. 进入"设置" → "正则清洗设置"（Regex Lab 标签页）
2. 点击底部橙色"重置为默认值"按钮
3. 确认弹窗提示
4. 等待页面自动刷新

### 6.3 首次初始化建议

如果遇到以下情况，建议执行重置操作：

1. **前端界面空白**: 配置文件损坏或格式错误
2. **AI 识别异常**: AI 规则配置不正确
3. **正则清洗失效**: 正则表达式配置错误
4. **版本升级后**: 确保使用最新的标准配置

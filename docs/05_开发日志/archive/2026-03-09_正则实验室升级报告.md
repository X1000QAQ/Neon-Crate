### 第二步：RegexLab 功能升级 ✅

**修改文件**: `frontend/components/settings/RegexLab.tsx`

**新增功能**:
1. **加载系统正则规则**: 页面加载时自动从后端获取当前配置
2. **正则规则编辑器**: 新增大型文本框用于编辑系统正则规则
3. **保存功能**: 新增"应用并保存到系统"按钮，将调试好的正则写入 config.json
4. **重置优化**: 重置后自动重新加载规则，无需刷新页面

**核心代码**:
```typescript
// 加载规则
const loadRegexRules = async () => {
  const config = await api.getSettings();
  setRegexRules(config.settings.filename_clean_regex || '');
};

// 保存规则
const handleSave = async () => {
  const config = await api.getSettings();
  config.settings.filename_clean_regex = regexRules;
  await api.updateSettings(config);
  alert('正则规则已保存到系统配置');
};

// 重置规则
const handleReset = async () => {
  await fetch('/api/v1/settings/reset', {
    method: 'POST',
    body: JSON.stringify({ target: 'regex' })
  });
  await loadRegexRules(); // 自动重新加载
};
```

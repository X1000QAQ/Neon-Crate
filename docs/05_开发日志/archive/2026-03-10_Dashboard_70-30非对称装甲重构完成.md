### 1. 小型装甲块（统计卡片）

**文件**: `frontend/app/globals.css` - `.cyber-armor-block-small`

**关键改造**：
```css
.cyber-armor-block-small {
  background: var(--black-color) !important;
  border: 10px solid var(--yellow-color) !important; /* 边框从 6px 增至 10px */
  
  /* 70/30 非对称切角逻辑 */
  clip-path: polygon(
    0% 0%,                      /* 左上角起点 */
    70% 0%,                     /* 顶部左侧 70% 突出 */
    calc(70% + 15px) 15px,      /* 右侧 30% 下沉 - 45° 切角 */
    100% 15px,                  /* 顶部右侧下沉平台 */
    100% calc(100% - 20px),     /* 右侧垂直边 */
    calc(100% - 20px) 100%,     /* 右下角 45° 切角 */
    calc(70% - 15px) 100%,      /* 底部右侧 30% 内凹起点 */
    calc(70% - 30px) calc(100% - 15px), /* 底部内凹 45° 切角 */
    20px calc(100% - 15px),     /* 底部左侧平台 */
    0% calc(100% - 35px)        /* 左下角 45° 切角 */
  );
  
  padding: 24px; /* 从 20px 增至 24px */
}
```

**视觉效果**：
- ✅ 边框视觉重量显著增加（10px 黄色装甲）
- ✅ 顶部左侧 70% 区域突出，形成"装甲肩部"

## 🚀 技术要点

### CSS clip-path 精确控制
- 使用 `calc()` 函数实现 70/30 比例
- 通过固定像素值（15px, 20px, 25px）控制 45° 切角
- 确保在不同容器尺寸下比例一致

### 字体体系
- **标题**：Advent Pro（科技感，字符间距 2px）
- **数值**：Hacked（黑客终端，字符间距 4px）
- **图标**：Lucide React（线条粗细 1.8）

### 响应式考虑
- 优先保证 1920x1080 分辨率下的比例协调
- 使用 Tailwind 响应式网格（md:grid-cols-2 lg:grid-cols-4）
- padding 和 gap 适配不同屏幕尺寸

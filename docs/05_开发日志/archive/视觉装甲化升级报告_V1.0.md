### 1️⃣ 字体体系重构（Typography Sync）

#### ✅ 全局科技感字体：Advent Pro

**改造位置**：`frontend/app/globals.css`

```css
/* 🎯 全局科技感字体：Advent Pro 作为主字体 */
body {
  font-family: 'Advent Pro', 'Orbitron', 'Segoe UI', sans-serif;
}

/* 🎯 全局标题强制使用 Advent Pro */
h1, h2, h3, h4, h5, h6 {
  font-family: 'Advent Pro', sans-serif !important;
  font-weight: 600;
  letter-spacing: 0.05em;
}

/* 🎯 所有标签、按钮文本使用 Advent Pro */
label, button, .btn, [role="button"] {
  font-family: 'Advent Pro', sans-serif !important;
  font-weight: 500;
  letter-spacing: 0.03em;
}
```

**效果**：
- ✅ 全站范围内应用 Advent Pro 科技字体
- ✅ 标题、标签、按钮呈现现代工业感
- ✅ 字符间距优化，提升可读性

### 3️⃣ 交互细节调整

#### ✅ 输入框聚焦：黄色霓虹呼吸光晕

**改造位置**：`frontend/app/globals.css`

```css
/* 🎯 通用输入框聚焦：黄色霓虹呼吸光晕 */
input:focus,
textarea:focus,
select:focus {
  outline: none !important;
  border-color: var(--cyber-yellow) !important;
  box-shadow: 
    0 0 0 1px var(--yellow-color-opacity),
    0 0 12px rgba(249, 240, 2, 0.4),      /* 黄色霓虹呼吸光晕 */
    0 0 24px rgba(249, 240, 2, 0.2) !important;
  animation: input-glow-pulse 2s ease-in-out infinite;
}

@keyframes input-glow-pulse {
  0%, 100% {
    box-shadow: 
      0 0 0 1px var(--yellow-color-opacity),
      0 0 12px rgba(249, 240, 2, 0.4),
      0 0 24px rgba(249, 240, 2, 0.2);
  }
  50% {
    box-shadow: 
      0 0 0 1px var(--yellow-color-opacity),
      0 0 16px rgba(249, 240, 2, 0.6),
      0 0 32px rgba(249, 240, 2, 0.3);
  }
}
```

**效果**：
- ✅ 输入框聚焦时出现黄色霓虹光晕
- ✅ 光晕呼吸动画（2秒周期）
- ✅ 边框颜色从青色切换为黄色

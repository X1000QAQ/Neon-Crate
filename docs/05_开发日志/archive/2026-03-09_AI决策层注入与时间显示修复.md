## 任务概述

本次任务将 AI Agent 决策层强行注入刮削流程，并修复创建时间显示异常和 IMDB 缺失问题。

### 核心目标

1. 在刮削流程中强制注入 AI 智能决策层
2. 修复前端创建时间显示异常
3. 恢复 IMDB 数据链路
4. 加固前端日期解析器

---

## 问题分析

### 问题 1：刮削精度不足

**现象**：
- 文件名如 "复仇者联盟4 终局之战 Avengers Endgame" 直接搜索 TMDB 失败
- 技术噪音（分辨率、编码格式等）干扰搜索结果

**根本原因**：
- 刮削流程直接使用 `clean_name` 搜索 TMDB
- 缺少 AI 智能提炼层，无法剥离技术噪音

### 问题 2：创建时间显示异常

**现象**：
- 前端显示 "格式错误" 或 "刚才"
- 数据库返回的时间格式不一致

**根本原因**：
- 后端 `db_manager.py` 返回字段名为 `time`，前端期待 `created_at`
- 后端对时间字符串进行了盲目替换（`.replace(" ", "T")`）

### 问题 3：IMDB 数据缺失

**现象**：
- 刮削成功后 `imdb_id` 字段为 `None`
- 无法用于字幕搜索和去重

**根本原因**：
- `tasks.py` 中强行将 `imdb_id` 设为 `None`
- 未尝试从 TMDB 详情中获取 IMDB ID

## 解决方案

### 第一步：注入 AI 智能决策层

**修改文件**: `backend/app/api/v1/endpoints/tasks.py`

#### 1.1 导入 AI Agent

```python
# 导入刮削引擎和 AI Agent
from app.services.metadata.adapters import TMDBAdapter
from app.services.ai.agent import AIAgent

scraper = TMDBAdapter(api_key=tmdb_api_key)
ai_agent = AIAgent(db)

logger.info("[AI] AI 决策层已激活，将对所有文件名进行智能提炼")
```

#### 1.2 强制注入 AI 分析

```python
# AI 决策层强制注入：禁止直接搜索 TMDB
logger.info(f"[AI] 调用 AI Agent 分析文件名: {clean_name or file_name}")
ai_result = await ai_agent.ai_identify_media(
    cleaned_name=clean_name or file_name,
    full_path=file_path,
    type_hint=media_type
)

if not ai_result:
    logger.warning(f"[AI] AI 分析失败，跳过任务 {task_id}")
    db.update_task_status(task_id=task_id, status="failed")
    failed_count += 1
    processed += 1
    continue

# 使用 AI 提炼的精准查询词
refined_query = ai_result.get("query", clean_name or file_name)
refined_year = ai_result.get("year", task.get("year"))
refined_type = ai_result.get("type", media_type).lower()

logger.info(f"[AI] AI 提炼结果: query='{refined_query}', year='{refined_year}', type='{refined_type}'")
```

#### 1.3 使用 AI 提炼结果搜索 TMDB

```python
# 使用 AI 提炼后的查询词搜索 TMDB
if refined_type == "movie":
    results = scraper.search_movie(
        query=refined_query,
        year=refined_year
    )
else:
    results = scraper.search_tv(
        query=refined_query,
        year=refined_year
    )
```

**效果**：
- AI 会将 "复仇者联盟4 终局之战 Avengers Endgame" 提炼为 "Avengers Endgame"
- 自动剥离技术噪音（分辨率、编码格式等）
- 提升 TMDB 匹配成功率

**修改文件**: `backend/app/infra/database/db_manager.py`

#### 2.1 修改字段名

**修改前**：
```python
"time": r[12].replace(" ", "T").replace(",", ".") if r[12] and " " in r[12] else r[12]
```

- 移除盲目的字符串替换，直接返回原始时间字符串

### 第四步：前端日期解析器加固

**修改文件**: `frontend/lib/utils.ts`

#### 4.1 彻底重写 formatDate

```typescript
export function formatDate(dateString: string): string {
  if (!dateString) return '刚刚';
  
  // 暴力修复：如果包含逗号或空格，直接强转
  const cleanStr = dateString.replace(' ', 'T').replace(',', '.');
  const date = new Date(cleanStr);
  
  return isNaN(date.getTime()) ? '格式错误' : date.toLocaleString('zh-CN', {
    year: 'numeric',
    month: '2-digit',
    day: '2-digit',
    hour: '2-digit',
    minute: '2-digit'
  });
}
```

**关键改进**：
- 简化逻辑，移除冗余判断
- 统一处理空格和逗号
- 兜底显示 "格式错误" 而非 "刚才"

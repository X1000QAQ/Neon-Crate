### 2️⃣ 容器"重装化"（Heavy Armor Container）

#### 🛡️ 黄色主装甲
- **取材对象**: `cyberpunk-2077.css` 的 `.cyberpunk.section` 逻辑
- **边框厚度**: 
  - 常规边框：`12px solid var(--yellow-color)`
  - 左侧装甲：`20px solid var(--yellow-color)`（重型加厚）
  - 顶部装甲：`15px solid var(--yellow-color)`

#### 🔋 非对称青色点缀（关键）
- **禁止**: 四边青色框
- **实现**: 使用 `::after` 伪元素在右侧创建垂直能量条
- **位置**: 右侧边缘，距离右边 10px
- **尺寸**: 宽度 6px，高度占容器 70%
- **效果**: 
  - 渐变背景（透明 → 霓虹绿 → 青色 → 霓虹绿 → 透明）
  - 霓虹发光：`box-shadow: 0 0 12px var(--cyber-cyan)`
  - 脉冲动画：2s 呼吸效果

```css
.heavy-armor-container::after {
  content: "";
  position: absolute;
  right: 10px;
  top: 50%;
  transform: translateY(-50%);
  width: 6px;
  height: 70%;
  background: linear-gradient(
    180deg,
    transparent 0%,
    var(--neon-color) 10%,
    var(--cyber-cyan) 50%,
    var(--neon-color) 90%,
    transparent 100%
  );
  box-shadow: 
    0 0 12px var(--cyber-cyan),
    0 0 25px rgba(0, 230, 246, 0.6);
  animation: energy-pulse 2s ease-in-out infinite;
}
```

#### 🔺 非对称斜角裁剪
- **取材**: 原版 `clip-path` 逻辑
- **效果**: 右上角和右下角斜切，模拟重型工业设备轮廓

```css
clip-path: polygon(
  0 0, 
  calc(100% - 25px) 0, 
  100% 25px, 
  100% calc(100% - 40px), 
  calc(100% - 40px) 100%, 
  0 100%
);
```

## 🎯 关键设计决策

### ✅ 非对称设计原则
- **黄色装甲**: 包围全身（四边）
- **青色光条**: 只准出现在右侧（禁止四边青色框）
- **理由**: 模拟游戏中的能量指示灯，避免过度对称导致的呆板感

### ✅ 颜色变量调用
所有颜色必须调用 `cyberpunk-2077.css` 定义的变量：
- `--yellow-color`: `#f9f002`（主装甲色）
- `--neon-color`: `#39ff14`（霓虹绿）
- `--cyber-cyan`: `#00e6f6`（青色点缀）
- `--border-color`: `#8ae66e`（辅助绿）
- `--red-color`: `#ff003c`（警告红）

### ✅ 动画周期统一
- 使用原版变量 `--glitched-duration: 0.9s` 和 `--glitched-long-duration: 3s`
- 确保所有故障动画、扫描线、呼吸光晕保持统一节奏

# Neon Crate - 视觉重塑：装甲模式激活

> **任务代号**: ARMOR MODE ACTIVATION  
> **执行日期**: 2026-03-10  
> **状态**: ✅ 已完成  
> **影响范围**: 前端全局样式系统 (globals.css)

## 一、任务背景

### 1.1 问题诊断

在 V1.0 发布后，用户反馈前端界面虽然已具备赛博朋克风格，但在以下方面仍显不足：

1. **边框装甲感不足**: 当前 `glass-effect` 边框厚度为 6px/18px，在大屏幕上显得单薄
2. **字体科技感欠缺**: 部分区域未完全应用 Advent Pro 科技字体
3. **密码输入框缺乏黑客感**: 未强制应用 Hacked 字体，字符间距过窄
4. **交互反馈不够强烈**: 输入框聚焦光晕、按钮悬停效果偏弱
5. **背景纹理过于微弱**: 点状矩阵和扫描线不够明显

### 1.2 设计目标

参考 `cyberpunk-2077风格设计/Neon-Crate_赛博朋克视觉风格指导文档.md`，执行以下升级：

- **装甲边框强化**: 主边框 6px → 8px，左侧装甲 18px → 24px
- **双重边框效果**: 内部青色边框 2px → 3px，增加复合装甲层叠感
- **字体体系重构**: 全站强制 Advent Pro，密码框强制 Hacked + 字符间距 3px
- **交互细节调整**: 输入框聚焦光晕增强，按钮悬停装甲光晕加强
- **背景纹理增强**: 点状矩阵从 1px → 1.5px，扫描线透明度提升

## 二、核心改造清单

### 2.1 字体体系重构 (Typography Sync)

#### ✅ 密码输入框黑客字体强制

```css
/* 密码输入框 & 敏感密钥区域：强制 Hacked 黑客字体 */
input[type="password"],
.hacked-font,
.secret-key-display {
  font-family: 'Hacked', monospace !important;
  letter-spacing: 3px !important; /* 字符间距拉开至 3px */
}
```

**改造效果**: 密码输入框字符间距从 2px 提升至 3px，黑客终端感显著增强

### 2.2 装甲边框强化 (Heavy Armor Borders)

#### ✅ 主容器装甲加厚

```css
.glass-effect {
  background: rgba(0, 0, 0, 0.5) !important;
  
  /* 装甲边框强化：主边框 8px + 左侧装甲 24px */
  border: 8px solid var(--cyber-yellow) !important;
  border-left: 24px solid var(--cyber-yellow) !important;
  
  /* 双重边框效果：内部青色间隙 + 复合装甲层叠感 */
  box-shadow: 
    inset 0 0 0 3px var(--cyber-cyan),
    inset 3px 3px 0 4px rgba(0, 0, 0, 0.4),
    12px 12px 0px rgba(249, 240, 2, 0.25),
    0 0 25px rgba(249, 240, 2, 0.35);
  
  /* 更激进的斜角裁剪 */
  clip-path: polygon(
    0 0, 
    calc(100% - 18px) 0, 
    100% 18px, 
    100% calc(100% - 35px), 
    calc(100% - 35px) 100%, 
    0 100%
  );
  
  backdrop-filter: blur(10px) !important;
}
```

**改造对比**:

| 属性 | 改造前 | 改造后 | 提升幅度 |
|------|--------|--------|----------|
| 主边框 | 6px | 8px | +33% |
| 左侧装甲 | 18px | 24px | +33% |
| 内部青色边框 | 2px | 3px | +50% |
| 斜角裁剪 | 12px/30px | 18px/35px | +50%/+17% |

### 2.3 交互细节调整 (Interaction Enhancement)

#### ✅ 输入框聚焦霓虹呼吸光晕

```css

textarea:focus,

}
```

**改造效果**: 聚焦光晕从 12px/24px 提升至 15px/30px，呼吸效果更强烈

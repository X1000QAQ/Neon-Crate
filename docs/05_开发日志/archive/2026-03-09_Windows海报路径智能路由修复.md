# Windows 海报路径智能路由修复

**日期**: 2026-03-09  
**类型**: 前端路径解析增强  
**影响范围**: MediaWall 组件海报显示逻辑

---

## 问题背景

在 Windows 开发环境下，本地海报路径解析存在以下问题：

1. **Docker 标准路径**：`/storage/Movies/Dune/Dune.mkv` 可以正常解析
2. **Windows 绝对路径**：`D:\test\media\Movies\Dune\Dune.mkv` 解析失败，导致海报裂图
3. **路径格式差异**：Windows 盘符、反斜杠、media 文件夹锚点未被正确处理

---

## 修复方案

### 核心逻辑：三级路径解析策略

在 `frontend/components/media/MediaWall.tsx` 的 `getPosterUrl` 函数中实现智能路径解析：

```typescript
const getPosterUrl = (task: Task): string => {
  if (task.status === 'archived' && task.target_path) {
    let dirPath = '';
    const tp = task.target_path;

    // 1. Docker AIO 标准模式：去除 /storage/ 前缀
    if (tp.startsWith('/storage') || tp.startsWith('\\storage')) {
      dirPath = tp.replace(/^[\/\\]storage[\/\\]/, '');
    }
    // 2. Windows 开发者模式兼容：利用 "media" 文件夹作为锚点进行切分
    else if (tp.includes('media')) {
      const parts = tp.split(/[/\\]/);
      const mediaIndex = parts.indexOf('media');
      if (mediaIndex !== -1) {
        // 提取 media 之后的所有路径片段
        dirPath = parts.slice(mediaIndex + 1).join('/');
      } else {
        dirPath = tp.replace(/^[a-zA-Z]:[/\\]/, ''); // 暴力清理盘符
      }
    }
    // 3. 终极兜底：清理 Windows 盘符和根斜杠
    else {
      dirPath = tp.replace(/^[a-zA-Z]:[/\\]/, '').replace(/^[\/\\]/, '');
    }
    // 去除末尾的具体文件名（如 .mkv），只保留文件夹路径
    const finalDir = dirPath.split(/[/\\]/).slice(0, -1).join('/');
    // 拼装最终经过 FastAPI 代理的资产请求路径
    return `/api/v1/assets/${finalDir}/poster.jpg`;
  }
  // 由于禁用了外部网络请求，未归档或计算失败统一显示默认占位图
  return '/placeholder-poster.jpg';
};
```

---

## 路径解析示例

### 场景 1：Docker AIO 标准模式
- **输入**：`/storage/Movies/Dune/Dune.mkv`
- **处理**：去除 `/storage/` 前缀
- **输出**：`/api/v1/assets/Movies/Dune/poster.jpg`

### 场景 2：Windows 开发者模式（含 media 锚点）
- **输入**：`D:\test\media\Movies\Dune\Dune.mkv`
- **处理**：定位 `media` 索引，提取后续路径
- **输出**：`/api/v1/assets/Movies/Dune/poster.jpg`

### 场景 3：Windows 绝对路径（无 media 锚点）
- **输入**：`C:\Videos\Dune\Dune.mkv`
- **处理**：清理盘符和根斜杠
- **输出**：`/api/v1/assets/Videos/Dune/poster.jpg`

---

## 技术要点

1. **正则表达式**：`/^[\/\\]storage[\/\\]/` 同时匹配正斜杠和反斜杠
2. **锚点切分**：使用 `media` 文件夹作为路径分割标记
3. **兜底策略**：清理 Windows 盘符 `[a-zA-Z]:[/\\]`
4. **路径标准化**：统一使用正斜杠 `/` 拼接最终 URL

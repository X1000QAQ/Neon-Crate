# 装甲化 CSS 变更摘要

> **执行日期**: 2026-03-10

## 核心变更对比表

### 1. 主容器装甲边框 (`.glass-effect`)

| 属性 | 改造前 | 改造后 | 变化 |
|------|--------|--------|------|
| `background` | `rgba(0,0,0,0.4)` | `rgba(0,0,0,0.5)` | 浓度 +25% |
| `border` | `6px` | `8px` | 厚度 +33% |
| `border-left` | `18px` | `24px` | 厚度 +33% |
| `padding` | `20px` | `24px` | +20% |
| `backdrop-filter` | `blur(8px)` | `blur(10px)` | +25% |
| 斜角裁剪 | `12px/30px` | `18px/35px` | +50%/+17% |

**box-shadow 变更**:
```css
/* 改造前 */
box-shadow: 
  inset 0 0 0 2px var(--cyber-cyan),
  10px 10px 0px rgba(249, 240, 2, 0.2),
  0 0 20px rgba(249, 240, 2, 0.3);

/* 改造后 */
box-shadow: 
  inset 0 0 0 3px var(--cyber-cyan),           /* +50% */
  inset 3px 3px 0 4px rgba(0, 0, 0, 0.4),      /* 新增：复合装甲层 */
  12px 12px 0px rgba(249, 240, 2, 0.25),       /* +20% */
  0 0 25px rgba(249, 240, 2, 0.35);            /* +25% */
```

/* 改造后 */

    inset 0 0 0 3px var(--cyber-yellow),

## 兼容性说明

### 向后兼容
- ✅ 所有现有组件无需修改即可享受装甲化效果
- ✅ 新增通用类为可选应用，不影响现有样式

### 性能影响
- ⚠️ `backdrop-filter: blur(10px)` 在低端设备可能影响性能
- ⚠️ 多层 `box-shadow` 叠加可能影响渲染性能
- 💡 建议在移动端添加降级方案

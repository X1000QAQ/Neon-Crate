# 国际化 - `lib/i18n.ts`

---

## 一、概述

轻量级 i18n 模块，支持中文（`zh`）和英文（`en`）双语。

---

## 二、核心函数

### `getTranslation(lang, key) -> string`

根据语言和 key 返回翻译文本，key 不存在时返回 key 本身作为兜底。

### `I18nKey` 类型

所有翻译 key 的联合类型，由 `translations` 对象的键推导。组件中 `t(key: I18nKey)` 可获得类型检查。

---

## 三、语言切换机制

```
用户在 BasicSettings 修改语言
  → SettingsHub.handleSave()
  → setLang(newLang)          # 更新 localStorage + dispatch CustomEvent
  → useLanguage 监听 languageChanged 事件
  → 重新渲染所有使用 t() 的组件
  → window.location.reload()  # 确保完整刷新（100ms 延迟）
```

---

## 四、注意事项

- 语言设置持久化在 `localStorage.ui_lang`
- 后端 `config.settings.ui_lang` 也存储语言偏好，两者通过保存设置同步
- 新增翻译 key 需同时在 `zh` 和 `en` 字典中添加，否则 TypeScript 会报错

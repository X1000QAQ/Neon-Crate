# lib/i18n.ts — 国际化字典

**文件路径**: `frontend/lib/i18n.ts`

---

## 职责

提供中英文双语字典和 `getTranslation()` 查询函数，支撑 `useLanguage` Hook 的 `t()` 方法。

---

## 结构

```typescript
type I18nKey = 'nav_dashboard' | 'nav_media' | 'stat_total_movies' | ...;

const translations: Record<Language, Record<I18nKey, string>> = {
  zh: { nav_dashboard: '仪表盘', ... },
  en: { nav_dashboard: 'Dashboard', ... },
};

export function getTranslation(lang: Language, key: I18nKey): string {
  return translations[lang]?.[key] ?? key;  // fallback 显示 key 本身
}
```

---

## 语言切换机制

```
用户在 BasicSettings 修改语言
  → setLang(newLang)         [useLanguage]
  → localStorage.setItem('ui_lang', newLang)
  → dispatchEvent(new CustomEvent('languageChanged'))

所有已挂载的 useLanguage 实例
  → 监听 'languageChanged' 事件
  → 重新读取 localStorage 更新 lang 状态
  → 组件重渲染，t() 返回新语言文本
```

## Key 命名规范

| 前缀 | 用途 |
|------|------|
| `nav_` | 导航标签 |
| `stat_` | 统计卡片标签 |
| `dashboard_` | 仪表盘相关 |
| `toolbar_` | 工具栏按钮 |
| `filter_` | 过滤器选项 |
| `status_` | 任务状态标签 |
| `ai_` | AI 相关提示 |
| `monitor_` | 监控视图 |
| `settings_` | 设置页面 |
| `alert_` / `confirm_` | 弹窗提示 |
| `btn_` | 通用按钮文本 |
| `ui_` | 通用 UI 文本 |

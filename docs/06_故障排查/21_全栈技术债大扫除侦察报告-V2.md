
**结论**：✅ 删除此误诊，`glob` 导入合法。

### ✅ 纠偏 2：Pydantic 模型空值是刻意设计

**原误诊**：问题 3.1 认为 `domain_system.py` 中的空值破坏 DRY 原则

**真相**：这是「单一真理来源」的正确实现：
- `default_config.py` 中定义真实默认值
- `domain_system.py` 中仅声明类型，赋空值
- `config_repo.py` 的 `get_all_config()` 动态合并，用 `DEFAULT_CONFIG` 填充空值

**流程**：
```
Pydantic 模型（类型声明）
  ↓
config_repo.get_all_config()
  ↓
遍历 DEFAULT_CONFIG，填充空值
  ↓
前端收到完整配置
```

**结论**：✅ 删除此误诊，设计正确。

### 病灶 2 [HIGH]：循环内重复定义常量

**位置**：`backend/app/api/v1/endpoints/tasks/scrape_task.py:975-976`

**当前代码**：
```python
    for f in resolved_meta.iterdir():
```

### 病灶 3 [MEDIUM]：XML 文件句柄泄漏

**位置**：`backend/app/api/v1/endpoints/tasks/scrape_task.py:1175-1180`

**当前代码**：
```python
        if nfo_path_check.exists():
            try:
                import xml.etree.ElementTree as ET
                _tree = ET.parse(nfo_path_check)  # ❌ 未使用 with，文件句柄可能泄漏
                _root = _tree.getroot()
                _existing_imdb = (_root.findtext("imdbid") or "").strip()
                if not _existing_imdb:
                    for _uid in _root.findall("uniqueid"):
                        if _uid.get("type") == "imdb":
                            _existing_imdb = (_uid.text or "").strip()
                            break
                if _existing_imdb and _existing_imdb == new_imdb_id.strip():
                    skip_metadata = True
                    logger.info(f"[REBUILD] 金标准校验通过: 本地已存在 IMDb ({new_imdb_id})，开启防爆护盾")
            except Exception as _e:
                logger.warning(f"[REBUILD] NFO 解析失败，防爆护盾离线: {_e}")
```

**问题**：
- `ET.parse()` 打开文件但未显式关闭
- 长期运行时可能导致文件描述符耗尽
- 在高并发场景下风险更高

**修复方案**：
```python
        if nfo_path_check.exists():
            try:
                import xml.etree.ElementTree as ET
                with open(nfo_path_check, 'r', encoding='utf-8') as f:
                    _tree = ET.parse(f)
                    _root = _tree.getroot()
                    _existing_imdb = (_root.findtext("imdbid") or "").strip()
                    if not _existing_imdb:
                        for _uid in _root.findall("uniqueid"):
                            if _uid.get("type") == "imdb":
                                _existing_imdb = (_uid.text or "").strip()
                                break
                    if _existing_imdb and _existing_imdb == new_imdb_id.strip():
                        skip_metadata = True
                        logger.info(f"[REBUILD] 金标准校验通过: 本地已存在 IMDb ({new_imdb_id})，开启防爆护盾")
            except Exception as _e:
                logger.warning(f"[REBUILD] NFO 解析失败，防爆护盾离线: {_e}")
```

**影响范围**：低（但长期风险高）

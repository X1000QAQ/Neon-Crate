## 一、现状全景

### 1.1 TV 元数据目录层级（当前逻辑）

```python
# scrape_task.py:1092
metadata_dir = os.path.dirname(os.path.abspath(target_path))
if re.match(r'^(Season|S)\s*\d+$|^Specials$', os.path.basename(metadata_dir), re.IGNORECASE):

```

**现有逻辑已经有 Season 跳升**：若 `target_path` 在 `Season X/` 子目录，`metadata_dir` 会自动跳升到剧集根目录。
**但问题存在于核级重置后**：
- `old_video_path` 取自递归 rglob，可能在 `Season 1/ep.mkv`
- `_rename_video_file` 重命名后文件仍留在 Season 子目录
- `metadata_dir` 被更新为 `os.path.dirname(new_video_path)`，退回到 Season 子目录
- 后续 NFO/poster 写入 Season 子目录 ❌

### 1.2 NFO/Poster 写入点（当前逻辑）

```python
# scrape_task.py:1218
nfo_filename = "tvshow.nfo" if body.media_type == "tv" else "movie.nfo"

```

**结论**：需要在核级重置更新 `metadata_dir` 后，对 TV 任务再次执行一次 Season 跳升。

### 1.3 目录重命名能力（当前：无）

当前 `manual_rebuild` 只能重命名视频**文件**，不能重命名**目录**（电影根目录 / 剧集根目录）。

需新增逻辑：
- 电影：若 `new_title (new_year)` 与目录名不符，重命名电影根目录
- 剧集：若 `season` 发生变化，创建新 `Season X/` 目录，移动视频，删除旧目录

## 三、修改方案（待点火）

### Phase 1 — 核级重置后 metadata_dir 再次 Season 跳升

```python
# 在 Step 3 同步 DB target_path 之后，追加：
metadata_dir = os.path.dirname(os.path.abspath(new_video_path))

if body.media_type == "tv" and re.match(
```

### Phase 2 — 目录重命名/搬运能力（目录土木工程）

**2a — 剧集 Season 目录重组（season 变更时）**
```python
# 在视频重命名（Step 2）之后、DB 同步之前：
if body.media_type == "tv" and task_season is not None:
    target_season_dir = Path(metadata_dir) / f"Season {task_season}"
    if not target_season_dir.exists():
        target_season_dir.mkdir(parents=True, exist_ok=True)
        logger.info(f"[NUCLEAR] 创建 Season 目录: {target_season_dir}")
    # 若视频不在目标 Season 目录，移动过去
    current_video = Path(new_video_path)
    if current_video.parent != target_season_dir:
        dest = target_season_dir / current_video.name
        shutil.move(str(current_video), str(dest))
        new_video_path = str(dest)
        logger.info(f"[NUCLEAR] 视频移动至: {dest}")
```

**2b — 根目录重命名（title/year 变更时）**
```python
# 在所有文件操作完成后，检查目录名是否需要更新：
if new_title and body.media_type == "movie":
    expected_dir_name = f"{new_title} ({new_year})" if new_year else new_title
    current_root = Path(metadata_dir)
    if current_root.name != expected_dir_name:
        new_root = current_root.parent / expected_dir_name
        if not new_root.exists():
            current_root.rename(new_root)
            metadata_dir = str(new_root)
            new_video_path = str(new_root / Path(new_video_path).name)
            # 同步 DB
            db.update_any_task_metadata(body.task_id, body.is_archive, target_path=new_video_path)
```

## 六、全链路流程示意图

```
扫描阶段（scan_task.py）
─────────────────────────────────────────────
ScanEngine.scan_directory()
  └─ file_info: {clean_name, year, season, episode, ...}
       ↓
  task_data 构建（clean_name 来自 MediaCleaner 正则）
       ↓
  [优先层] NFO Shift-Left
    ├─ 有 NFO + tmdb_id → task_data[clean_name] = nfo.title  ← 最高优先级
    │                     task_data[year]       = nfo.year
    │                     task_data[status]     = "archived"
    │                     has_nfo = True
    └─ 无 NFO → has_nfo = False
       ↓
  [兜底层] 失忆救援（仅 has_nfo=False 且 default_status=="archived"）
    └─ 从目录名提取 clean_name/year
       ↓
  [通用层] 本地海报绑定（仅 status=="archived"）
    └─ 扫描同级目录 poster.jpg / poster.png
       ↓
  db.insert_task(task_data)

刮削阶段（scrape_task.py）
─────────────────────────────────────────────
  task = db.get_tasks_needing_scrape()
       ↓
  [短路层] NFO 短路拦截
    ├─ 有 NFO + tmdb_id → db.update_task_status(archived)  ← continue 跳过 AI
    └─ 无 NFO → 继续
       ↓
  MediaCleaner.clean_name() → cleaned_filename
       ↓
  ai_agent.ai_identify_media(cleaned_filename, full_path, type_hint)
    └─ 返回: {query, year, type, season, episode}
       ↓
  refined_query = ai_result.get("query") or clean_name  ← 兜底
  refined_year  = ai_result.get("year")  or task.year
  refined_type  = 路径权威 > AI 建议
       ↓
  TMDB 搜索 → 归档
```

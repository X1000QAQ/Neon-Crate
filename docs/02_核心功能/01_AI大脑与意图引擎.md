# AI 大脑：意图识别与对话引擎

---

## 一句话理解

你跟它说「帮我扫描新文件」，它能听懂你的意思，自动执行扫描任务，并用你设定的人格风格回复你。这是整个系统的「大脑」。

---

## 它做了什么？

```
你说的话
  ↓
AI 理解意图（ACTION_SCAN / ACTION_SCRAPE / DOWNLOAD / 闲聊...）
  ↓
自动触发对应任务（扫描 / 刮削 / 下载）
  ↓
用你设定的人格风格回复
```

---

## 支持的意图

| 你说 | AI 识别为 | 自动执行 |
|---|---|---|
| 「扫描新文件」 | `ACTION_SCAN` | 触发物理扫描 |
| 「刮削元数据」 | `ACTION_SCRAPE` | 触发全量刮削 |
| 「补全字幕」 | `ACTION_SUBTITLE` | 触发字幕下载 |
| 「汇报战况」 | `SYSTEM_STATUS` | 返回真实任务统计+磁盘占用 |
| 「下载流浪地球」 | `DOWNLOAD` | 调用 Radarr/Sonarr 下载 |
| 「库里有没有三体」 | `LOCAL_SEARCH` | 搜索本地媒体库 |
| 「今天天气真好」 | `CHAT` | 普通闲聊 |

---

## 全时态感知：它知道现在的状态

每次对话前，系统自动注入真实数据：

```
总任务数 / 已归档 / 待处理 / 失败数 / 磁盘占用率 / 最近30条日志
```

「汇报战况」时，AI 被强制要求只能使用这些真实数据回答，禁止编造数字。

---

## 人格可以自定义

在「AI 人格设定」页面，你可以设置：

- **AI 名称**：叫什么名字
- **人格设定**（ai_persona）：「专业管家」还是「二次元萌妹」
- **意图路由规则**（master_router_rules）：教 AI 怎么识别你的指令
- **归档专家规则**（expert_archive_rules）：教 AI 怎么从文件名提取片名

这些规则都存在数据库里，修改后**实时生效**，不需要重启服务。

---

## 双引擎支持

| 引擎 | 配置 | 说明 |
|---|---|---|
| 云端 API | `llm_cloud_*` | OpenAI / DeepSeek 等兼容接口 |
| 本地 Ollama | `llm_local_*` | 本地部署的 Ollama 服务 |

在「推理引擎」设置页切换，切换后立即生效。

---

## 降级保护

AI 挂了/超时/API Key 没填 → 自动降级为关键词匹配，系统不会因 AI 失联而瘫痪。

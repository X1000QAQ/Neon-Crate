## 四、核心数据流

### 4.1 普通请求链路

```
浏览器
  └─► Next.js (frontend/lib/api.ts)
        └─► FastAPI (backend/app/api/v1/endpoints/)
              └─► Service 层 (backend/app/services/)
                    └─► SQLite / 外部 API（TMDB / OpenSubtitles）
```

### 4.2 AI 流式响应链路

```
AiSidebar.tsx
  └─► lib/api.ts (fetch stream)
        └─► POST /api/v1/agent (endpoints/agent.py)
              └─► services/ai/agent.py (意图路由)
                    ├─► services/ai/llm_client.py (LLM 流式调用)
                    ├─► services/downloader/servarr.py (下载指令)
                    └─► SSE chunk → 逐字回流前端
```

### 4.3 认证链路

```
app/auth/login/page.tsx
  └─► lib/api.ts
        └─► POST /api/v1/auth/login (api/auth.py)
              └─► infra/security/crypto.py (bcrypt 校验 + JWT 签发)
                    └─► Token → localStorage
                          └─► AuthGuard.tsx 校验后放行路由
```

### 4.4 归档全链路

```
触发扫描 (POST /scan)
  └─► services/scraper/engine.py (目录遍历 + 格式/大小过滤)
        └─► services/scraper/cleaner.py (文件名清洗)
              └─► db_manager.py (入库 tasks 表，去重拦截)

触发刮削 (POST /scrape_all)
  └─► services/metadata/adapters.py (TMDB 搜索)
        └─► services/metadata/metadata_manager.py (NFO + 海报 + Fanart)
              └─► services/organizer/hardlinker.py (SmartLink 归档)
                    └─► db_manager.py (更新 media_archive 表)

触发字幕 (POST /find_subtitles)
  └─► services/subtitle/engine.py
        ├─► 本地字幕检测（已存在则跳过，不调用 API）
        └─► OpenSubtitles API → 下载 .ai.zh-CN.srt
```

## 五、技术栈速览

| 层次 | 技术 |
|------|------|
| 前端框架 | Next.js 14 (App Router) + TypeScript |
| 前端样式 | Tailwind CSS + 自定义 CSS 变量（霓虹主题）|
| 前端组件 | React + lucide-react 图标 |
| 国际化 | 自研 i18n（中/英双语） |
| 后端框架 | FastAPI 0.104.1 (Python 3.12) |
| 数据库 | SQLite（WAL 模式） |
| AI 接入 | OpenAI 兼容 API（流式 SSE）|
| 认证 | JWT + Fernet 加密（bcrypt 密码哈希）|
| 外部 API | TMDB（元数据）/ OpenSubtitles（字幕）/ Radarr+Sonarr（下载）|
| 容器化 | Docker + docker-compose |
